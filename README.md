# Entanglement-in-prepare-and-measure-scenarios-without-receiver-inputs

Here we provide the optimal entanglement-assisted classical communication protocols for the minimal and next-to-minimal scenarios discussed in the paper titled "Entanglement-in-the-prepare-and-measure-scenario". 


MAIN FILE: 

EACCprotocol: computes the violation of the entanglment-assisted classical communication protocols S1,S2 and S3


DATA FILES: 

S1_protocols.mat

S2_protocols.mat

S3_protocols.mat

Data containing a list of the following format: (obj,B,A,rho).

obj - the optimal value

B -  Bob's measurements

A - Alice's measurement

rho - shared entangled state
